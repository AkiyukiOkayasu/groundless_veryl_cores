LpfShiftSigned

ビットシフトによる一次ローパスフィルタ (指数移動平均、符号付き版) 乗算器を使用せず、シフト演算のみで実装

伝達関数: y[n] = y[n-1] + (x[n] - y[n-1]) >> SHIFT

カットオフ周波数の近似: fc ≈ fs / (2π * 2^SHIFT) 例: fs=48kHz, SHIFT=8 → fc ≈ 30Hz

SHIFT値が大きいほどカットオフ周波数が低くなり、より滑らかに補間される。 SHIFTはDATA_WIDTHより大きくても問題ない。

Parameters

DATA_WIDTH u32 入出力データ幅
SHIFT u32 シフト量 (1以上)。大きいほど低いカットオフ周波数

Ports

clk input clock システムクロック
rst input reset リセット (出力を0に初期化)
enable input bbool イネーブル信号 (trueでフィルタ更新)
x input signedlogic 入力サンプル (符号付き)
y output signedlogic フィルタ出力 (符号付き)

MultiWaveCore

マルチ波形生成コアモジュール 位相入力から4種類の波形（Sine, Triangle, Saw, Pwm）を生成し、選択出力 Sineは線形補間付きLUT、他は組み合わせ回路 レイテンシ整合のため、全波形を1クロック遅延させて出力 レイテンシー: 1クロック

Ports

clk input clock クロック
phase input Phase phase_t 位相入力 (0 ~ 2^32-1)
wave_type input WaveTypes WaveType 波形タイプ選択
duty_cycle input Phase phase_t PWMデューティ比 (Pwm以外では無視)
pcm_out output FixedPoint q8_24 出力波形 (Q8.24形式)

MultiWaveOscillator

波形選択式オシレーター 4種類の波形（Sine, Triangle, Saw, Pwm）を切り替え可能 Sineは線形補間付きLUT、他は組み合わせ回路 波形切り替え時のレイテンシ整合のため、全波形を1クロック遅延させて出力 レイテンシー: 1クロック

Ports

clk input clock クロック
rst input reset リセット 位相ゼロリセット
phase_step input Phase step_t 1クロックあたりの位相ステップ量
wave_type input WaveTypes WaveType 波形タイプ選択
duty_cycle input Phase phase_t PWMデューティ比 (Pwm以外では無視)
pcm_out output FixedPoint q8_24 出力波形 (Q8.24形式)

Phasor

位相発生器 全ての波形オシレーターの基礎となるモジュール 負の周波数（位相の逆回転）にも対応

Ports

clk input clock システムクロック
rst input reset 位相ゼロリセット
phase_step input Phase step_t 1クロックあたりの位相ステップ量: step = (ターゲット周波数 * 2^48) / システムクロック周波数
phase output Phase phase_t 位相出力 (0 ~ 2^32-1)
